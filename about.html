About This Site

Check out my art! Is it good?

This website was created as a fictional character gallery project. The theme focuses on fantasy characters because fantasy worlds allow creators to explore unique powers, cultures, and adventures.

The design uses a dark fantasy style with card layouts to make each character easy to view. The goal was to create a clean, organized gallery that could eventually be expanded into a larger character database.

All characters featured on this website are fictional creations.
